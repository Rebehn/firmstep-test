# firmstep-test
###Built Using
- HTML
- CSS
- Javascript
- Express
- Node.js
- MongoDB
###Using
To use make sure you have both Node and npm installed then:
```
$ git clone git@github.com:Rebehn/firmstep-test.git
$ cd firmstep-test
$ npm i
$ nodemon
```
Then open up your browser and navigate to localhost:8000 to view.
